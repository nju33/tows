A tool for Node.js project to install the same dependencies of its project to each workspace package

## Usage

### Rust

```bash
cargo install tows

tows
```

### Node

```bash
npx @nju33/tows
```

### Options

#### `--cwd,-C`

Directory name which `package.json`(default) will be placed.

#### `--filename,-f`

Target filename. (default `package.json`)
